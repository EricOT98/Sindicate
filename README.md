# Sindicate
A 2d shoot'em up platformer build with C++ and SDL2. Winner of best AI award at GamesFleadh 2019.
Made during a 4 week period as part of 4th year ITC games development course.

Programmers:  
* Liam Hickey
* Eric O' Toole
* Dylan Murphy
* Michael Bridgette

Artists:  
* Billie Jean Doheny
* Angela Young
